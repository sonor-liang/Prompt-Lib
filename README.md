## 使用方式

进入对应职业目录，每个 `.md` 文件是一个提示词模板，直接复制使用即可。

## 收录标准

高质量的提示词模板应具备：
- ✅ 经过实际验证，效果良好
- ✅ 可复用性强，适用范围广
- ✅ 结构清晰，易于理解和使用
- ✅ 经过社区广泛采用
